War Card Game:
Using Java, I created a simulation of the card game 'War' which utilizes the four object-oriented programming principles of abstraction, encapsulation, polymorphism, and inheritance. This game can be played with a modified rule set according to Promineo Tech's guidelines with War.java or the traditional (harder to simulate) rule set on WarTwoPointO.java.

Jeep Sales Spring Boot Application:
This is a standalone application using the Java-based framework of Spring Boot. With these tools, I created a backend system for a hypothetical Jeep model SQL database that can be accessed by outside users via browser. The application ensures no malicious or problematic data is able to affect the Jeep database.
